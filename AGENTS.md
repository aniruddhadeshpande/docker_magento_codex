# Agent Instructions

This repository is used for a Magento 2 / Adobe Commerce learning project.

## Working Style

- Explain every phase before coding.
- Implement one phase at a time.
- Do not skip acceptance criteria.
- Do not implement future phases early.
- Keep code simple because this is a learning project.

## Magento Rules

- Do not modify Magento core files.
- Do not use ObjectManager directly.
- Prefer dependency injection.
- Prefer Magento service contracts where reasonable.

## Phase Completion

After each phase:

- Show files changed.
- Provide manual test commands.
- Explain what was learned.
